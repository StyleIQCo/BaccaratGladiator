# QR Code Usage

Generated QR code target:
- `https://www.baccaratgladiator.com/download`

Files:
- `qr-baccarat-download.png` (1400x1400)
- `qr-baccarat-download.svg` (vector)

## Before Launch
1. Update `IOS_APP_URL` and `ANDROID_APP_URL` in `download.html`.
2. Deploy `download.html` and the QR files.
3. Test QR scan on iPhone and Android.

## Optional Tracking
Use campaign params in ad placements, for example:
- `https://www.baccaratgladiator.com/download?utm_source=flyer&utm_medium=qr&utm_campaign=launch`

If you want forced QA testing:
- iOS force: `/download?platform=ios`
- Android force: `/download?platform=android`
